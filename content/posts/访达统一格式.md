## 终端输入指令

```bash
sudo find / -name ".DS_Store" -delete
```

<font color="#4bacc6">•	sudo</font>

{{< callout type="info" >}}
以 root 权限执行，可访问和修改整个系统。
{{< /callout >}}

<font color="#4bacc6">•	find /</font>

{{< callout type="info" >}}
从 根目录 / 开始递归扫描整个文件系统（包括系统目录、外接磁盘、受保护路径）。
{{< /callout >}}

<font color="#4bacc6">•	-name ".DS_Store"</font>

{{< callout type="info" >}}
匹配所有名为 .DS_Store 的文件（macOS Finder 生成的隐藏文件）。
{{< /callout >}}

<font color="#4bacc6">•	-delete</font>

{{< callout type="danger" >}}
直接删除匹配到的文件，无确认、无回收站、不可恢复。
{{< /callout >}}

## 重置访达

摁住 option 右键选择访达重新开始即可。
